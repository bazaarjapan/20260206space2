# HELIOS ATELIER - Solar System Studio

売れる体験を意識した、シネマティックな太陽系インタラクションシミュレーター

## 概要

HELIOS ATELIERは、Three.jsを使用して構築された3D太陽系シミュレーターです。リアルタイムで惑星の軌道を表示し、インタラクティブに操作できるビジュアル体験を提供します。

## 主な機能

- **3D太陽系シミュレーション**: 太陽と惑星のリアルタイム表示
- **インタラクティブなカメラ操作**:
  - 左ドラッグ: 回転
  - ホイール: ズーム
  - 右ドラッグ: 平行移動
- **時間倍率コントロール**: シミュレーション速度を0.1倍〜50倍まで調整可能
- **シネマモード**: 自動カメラアニメーション
- **フォーカス機能**: 特定の惑星にカメラをフォーカス
- **軌道ライン表示**: 惑星の軌道を可視化
- **惑星情報パネル**: 各惑星の詳細情報を表示
- **ポストプロセッシング**: UnrealBloomを使用したビジュアルエンハンス

## 技術スタック

- **Three.js**: 3Dグラフィックスレンダリング
- **OrbitControls**: カメラコントロール
- **EffectComposer & UnrealBloomPass**: ポストエフェクト
- **HTML5 / CSS3**: UI構築
- **JavaScript (ES6+)**: アプリケーションロジック

## 使い方

1. プロジェクトをローカルにクローン
2. ローカルWebサーバーを起動（例: `npx vite` or `python -m http.server`）
3. ブラウザでアクセス

### 操作方法

- **カメラ操作**: マウスドラッグとホイールで視点を変更
- **一時停止/再開**: 「一時停止」ボタンで制御
- **シネマモード**: 自動的に惑星を巡るカメラアニメーション
- **時間倍率**: スライダーで時間の流れを調整
- **フォーカス**: ドロップダウンから惑星を選択して追跡
- **軌道/ラベル**: チェックボックスで表示/非表示を切り替え
